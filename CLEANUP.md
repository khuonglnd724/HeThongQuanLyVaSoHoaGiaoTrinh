# Cleanup temporary/unnecessary files

## Safe to Delete (regenerated on build)
- **target/** folders (~435MB)
  - Java build outputs, classes, compiled JARs
  - Regenerated by `mvn clean package`

- **__pycache__/** folders (Python cache)
  - AI Service cache files
  - Regenerated automatically

## Safe to Delete (documentation only)
- **.github/** - Optional GitHub workflows
- **.vscode/** - IDE settings (use global settings)

## Do NOT Delete
- **src/** - Source code
- **pom.xml** - Build configs
- **docker-compose.yml** - Container orchestration
- **config-repo/** - Configuration files
- **init-scripts/** - Database scripts
- **.git/** - Version control

## Cleanup Commands

### Remove all build artifacts
```powershell
# PowerShell
Get-ChildItem -Recurse -Directory -Include "target" | Remove-Item -Recurse -Force
```

```bash
# Linux/Mac
find . -type d -name "target" -exec rm -rf {} + 2>/dev/null
```

### Remove Python cache
```powershell
# PowerShell
Get-ChildItem -Recurse -Directory -Include "__pycache__" | Remove-Item -Recurse -Force
```

```bash
# Linux/Mac
find . -type d -name "__pycache__" -exec rm -rf {} + 2>/dev/null
```

### Full cleanup (before committing)
```powershell
# PowerShell - Remove build and cache
$folders = @("target", "__pycache__", ".pytest_cache")
$folders | ForEach-Object { 
    Get-ChildItem -Recurse -Directory -Include $_ | 
    Remove-Item -Recurse -Force -ErrorAction SilentlyContinue
}
Write-Host "Cleanup complete!"
```

### Storage saved
- Before: ~435MB build artifacts
- After: Cleaned
- Rebuild time: ~3-5 minutes per service
